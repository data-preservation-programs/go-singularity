# 管理员命令

{% code fullWidth="true" %}
```
名称:
   singularity admin - 管理员命令

用法:
   singularity admin command [命令选项] [参数...]

命令:
   init              初始化数据库
   reset             重置数据库
   migrate-dataset   从旧的singularity mongodb迁移数据集
   migrate-schedule  从旧的singularity mongodb迁移计划
   help, h           显示命令列表或一个命令的帮助

选项:
   --help, -h  显示帮助信息
```
