# 使用现有的访问授权。

{% code fullWidth="true" %}
```
名称：
   singularity storage update storj existing - 使用现有的访问授权。

用法：
   singularity storage update storj existing [命令选项] <名称|ID>

描述：
   --access-grant
      访问授权。

选项：
   --access-grant value  访问授权。[$ACCESS_GRANT]
   --help, -h            显示帮助信息
```
