# 列出所有交易

{% code fullWidth="true" %}
```
名称：
   singularity deal list - 列出所有交易

用法：
   singularity deal list [命令选项] [参数...]

选项：
   --preparation value [ --preparation value ]  通过准备工作ID或名称筛选交易
   --source value [ --source value ]            通过源存储ID或名称筛选交易
   --schedule value [ --schedule value ]        通过时间表筛选交易
   --provider value [ --provider value ]        通过提供商筛选交易
   --state value [ --state value ]              通过状态筛选交易：proposed, published, active, expired, proposal_expired, slashed
   --help, -h                                   显示帮助信息
```
