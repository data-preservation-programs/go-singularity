# 安排交易

{% code fullWidth="true" %}
```
名称:
   singularity deal schedule - 安排交易

使用方法:
   singularity deal schedule command [命令选项] [参数...]

命令:
   create   创建一个计划，以发送交易给存储提供商
   list     列出所有交易安排计划
   update   更新现有计划
   pause    暂停指定的计划
   resume   恢复指定的计划
   help, h  显示命令列表或某个命令的帮助信息

选项:
   --help, -h  显示帮助信息
```
