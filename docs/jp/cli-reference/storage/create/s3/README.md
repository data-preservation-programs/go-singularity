```
NAME:
   singularity storage create s3 - AWS、Alibaba、Ceph、China Mobile、Cloudflare、ArvanCloud、DigitalOcean、Dreamhost、Huawei OBS、IBM COS、IDrive e2、IONOS Cloud、Liara、Lyve Cloud、Minio、Netease、RackCorp、Scaleway、SeaweedFS、StackPath、Storj、Tencent COS、Qiniu、Wasabiを含むAmazon S3互換のストレージプロバイダー

使用法:
   singularity storage create s3 コマンド [コマンドオプション] [引数...]

コマンド:
   aws           Amazon Web Services（AWS）S3
   alibaba       Alibaba Cloud Object Storage System（OSS）（以前のAliyun）
   arvancloud    Arvan Cloud Object Storage（AOS）
   ceph          Cephオブジェクトストレージ
   chinamobile   China Mobile Ecloud Elastic Object Storage（EOS）
   cloudflare    Cloudflare R2ストレージ
   digitalocean  DigitalOcean Spaces
   dreamhost     Dreamhost DreamObjects
   huaweiobs     Huaweiオブジェクトストレージサービス
   ibmcos        IBM COS S3
   idrive        IDrive e2
   ionos         IONOS Cloud
   liara         Liaraオブジェクトストレージ
   lyvecloud     Seagate Lyve Cloud
   minio         Minioオブジェクトストレージ
   netease       Neteaseオブジェクトストレージ（NOS）
   other         その他のS3互換プロバイダー
   qiniu         Qiniuオブジェクトストレージ（Kodo）
   rackcorp      RackCorpオブジェクトストレージ
   scaleway      Scalewayオブジェクトストレージ
   seaweedfs     SeaweedFS S3
   stackpath     StackPathオブジェクトストレージ
   storj         Storj（S3互換ゲートウェイ）
   tencentcos    Tencent Cloudオブジェクトストレージ（COS）
   wasabi        Wasabiオブジェクトストレージ
   help, h       コマンドの一覧を表示するか、コマンドのヘルプを表示する

オプション:
   --help, -h  ヘルプを表示する
```
