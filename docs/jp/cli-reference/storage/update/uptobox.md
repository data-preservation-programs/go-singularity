# Uptobox

{% code fullWidth="true" %}
```
名前:
   singularity storage update uptobox - Uptobox

使用法:
   singularity storage update uptobox [command options] <name|id>

説明:
   --access-token
      アクセストークンです。
      
      [https://uptobox.com/my_account](https://uptobox.com/my_account)から取得してください。

   --encoding
      バックエンドのエンコーディングです。
      
      詳細については[概要のエンコーディングセクション](/overview/#encoding)を参照してください。


オプション:
   --access-token value  アクセストークンです。 [$ACCESS_TOKEN]
   --help, -h            ヘルプを表示する

   上級オプション

   --encoding value  バックエンドのエンコーディングです。 (デフォルト: "Slash,LtGt,DoubleQuote,BackQuote,Del,Ctl,LeftSpace,InvalidUtf8,Dot") [$ENCODING]

```
