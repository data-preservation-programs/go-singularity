# レプリケーション / 契約管理

{% code fullWidth="true" %}
```
NAME:
   singularity deal - レプリケーション / 契約管理

USAGE:
   singularity deal command [command options] [arguments...]

COMMANDS:
   schedule     ディールをスケジュールする
   send-manual  マニュアルのディール提案を送信して、ブーストやレガシー市場に貢献する
   list         全てのディールを一覧表示する
   help, h      コマンドの一覧を表示するか、特定のコマンドのヘルプを表示する

OPTIONS:
   --help, -h  ヘルプを表示する
```
