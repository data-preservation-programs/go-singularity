# ウォレットの管理

{% code fullWidth="true" %}
```
NAME:
   singularity wallet - ウォレットの管理

USAGE:
   singularity wallet command [command options] [arguments...]

COMMANDS:
   import   エクスポートされたプライベートキーからウォレットをインポートする
   list     インポートされたすべてのウォレットをリストする
   remove   ウォレットを削除する
   help, h  コマンドの一覧または特定のコマンドのヘルプを表示する

OPTIONS:
   --help, -h  ヘルプを表示する
```
