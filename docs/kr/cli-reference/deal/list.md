# 거래 목록 확인

{% code fullWidth="true" %}
```
이름:
   singularity deal list - 모든 거래 목록 확인

사용법:
   singularity deal list [command options] [arguments...]

옵션:
   --preparation value [ --preparation value ]  준비 ID 또는 이름으로 거래 필터링
   --source value [ --source value ]            소스 저장소 ID 또는 이름으로 거래 필터링
   --schedule value [ --schedule value ]        일정으로 거래 필터링
   --provider value [ --provider value ]        공급자로 거래 필터링
   --state value [ --state value ]              상태로 거래 필터링: 제안됨, 게시됨, 활성, 만료됨, 제안 만료, 삭제됨
   --help, -h                                   도움말 표시
```
