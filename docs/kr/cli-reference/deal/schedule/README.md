# 거래 일정

{% code fullWidth="true" %}
```
이름:
   singularity deal schedule - 거래 일정 관리

사용법:
   singularity deal schedule command [command options] [arguments...]

COMMANDS:
   create   스토리지 제공자에 대한 거래 일정을 생성합니다.
   list     거래 일정 목록을 조회합니다.
   update   기존의 거래 일정을 수정합니다.
   pause    특정 거래 일정을 일시 정지합니다.
   resume   특정 거래 일정을 재개합니다.
   help, h  명령어 목록 또는 특정 명령어에 대한 도움말을 표시합니다.

옵션:
   --help, -h  도움말 표시
```
