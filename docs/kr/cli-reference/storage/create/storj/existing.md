# 기존 액세스 승인서를 사용하세요.

{% code fullWidth="true" %}
```
이름:
   singularity storage create storj existing - 기존 액세스 승인서를 사용하세요.

사용법:
   singularity storage create storj existing [command options] [arguments...]

설명:
   --access-grant
      액세스 승인서.


옵션:
   --access-grant value  액세스 승인서. [$ACCESS_GRANT]
   --help, -h            도움말 표시

   일반

   --name value  저장소의 이름 (기본값: 자동 생성)
   --path value  저장소의 경로

```
