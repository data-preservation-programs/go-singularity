# Put.io

{% code fullWidth="true" %}
```
이름:
   singularity storage create putio - Put.io

사용법:
   singularity storage create putio [명령 옵션] [인자...]

설명:
   --encoding
      백엔드의 인코딩입니다.
      
      자세한 내용은 [개요에서의 인코딩 섹션](/overview/#encoding)을 참조하세요.


옵션:
   --help, -h  도움말 표시

   고급

   --encoding value  백엔드의 인코딩입니다. (기본값: "Slash,BackSlash,Del,Ctl,InvalidUtf8,Dot") [$ENCODING]

   일반

   --name value  스토리지의 이름입니다. (기본값: 자동 생성)
   --path value  스토리지의 경로입니다.

```
