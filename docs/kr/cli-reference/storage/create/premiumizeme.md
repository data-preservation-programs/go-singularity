   --help, -h       도움말 표시

   고급

   --encoding value  백엔드의 인코딩입니다. (기본값: "Slash,DoubleQuote,BackSlash,Del,Ctl,InvalidUtf8,Dot") [$ENCODING]

   일반

   --name value  저장소의 이름 (기본값: 자동 생성)
