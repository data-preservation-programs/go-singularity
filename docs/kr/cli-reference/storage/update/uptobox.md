# Uptobox

{% code fullWidth="true" %}
```
이름:
   singularity storage update uptobox - Uptobox

사용법:
   singularity storage update uptobox [명령 옵션] <이름|ID>

설명:
   --access-token
      액세스 토큰입니다.
      
      https://uptobox.com/my_account에서 얻을 수 있습니다.

   --encoding
      백엔드의 인코딩입니다.
      
      자세한 내용은 [개요의 인코딩 섹션](/overview/#encoding)을 참조하십시오.


옵션:
   --access-token value  액세스 토큰입니다. [$ACCESS_TOKEN]
   --help, -h            도움말 표시

   고급

   --encoding value  백엔드의 인코딩입니다. (기본값: "슬래시,크기가 작은부등호,큰따옴표,역따옴표,삭제,제어,왼쪽 공백,유효하지 않은 UTF-8,닷") [$ENCODING]
```
